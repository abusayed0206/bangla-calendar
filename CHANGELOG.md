## [0.0.2] - 2025-12-21

### Added
- বাংলা তারিখ উইজেট (Bangla date widget)
- বাংলা ক্যালেন্ডার/পুঞ্জিকা (Bangla calendar/Punjika)
- সিস্টেম ট্রে সাপোর্ট (System tray support)
- স্বয়ংক্রিয় শুরু অপশন (Autostart option)
- ঋতু প্রদর্শন (Season display)
- Modern UI with Bangladesh flag green theme (#006A4E)
- Font caching for improved performance
- MSIX package for Windows Store

### Technical
- Built with Rust 2024 edition
- Windows crate v0.61 for native Win32 APIs
- Double buffering for flicker-free rendering
- Modular code architecture
